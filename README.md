# Keeper App

## 📌 Overview
A React-based note keeping application inspired by Google Keep that allows users to create and delete notes dynamically.

## 🛠 Tech Stack
- React
- JavaScript
- CSS

## ✨ Features
- Add notes
- Delete notes
- Component-based architecture
- State management using React hooks

## 🚀 How to Run
1. Clone the repository
2. Run `npm install`
3. Run `npm start`
